### Afternoon session ‚Äî Tracer bullet pipeline

Implemented the full `src/tracer_bullet/` package:

- **`signal.py`** ‚Äî Three swappable interaction signal strategies:
  - `build_threshold_signal`: ratings ‚â• threshold ‚Üí `interaction=1`; rows below threshold are **dropped** (not encoded as 0, to avoid falsely asserting negative preference)
  - `build_weighted_signal`: min-max normalizes rating to `[0, 1]` ‚Äî graded soft signal
  - `build_time_decay_signal`: weights each event by `exp(-Œª √ó days_since)` ‚Äî recent events score higher
  - `build_interactions` dispatcher + `train_test_split_by_time` (temporal split, no leakage)

- **`evaluation.py`** ‚Äî Ranking metrics for signal comparison:
  - `hit_at_k`: binary ‚Äî did any relevant item appear in top-K?
  - `ndcg_at_k`: position-sensitive DCG/IDCG ‚Äî earlier rank = higher score
  - `evaluate`: mean + std per metric across users (std is key for fairness across users)
